Основные типы данных
1. int 'Целые числа' - 1, 2, 3...
2. float 'Число с плавающей точкой' - 1.1, 1.2, 3.0...
3. str 'Строки'(последовательность символов) - 'Hello, world'...
4. list 'Списки'(последовательность объектов) - ['Hello, world', 1.1, 2.0]
5. dict 'Словари' (Список пар ключ-значение) - {'john' : '+1-23-45'}
6. tuple 'КортЕжи' (последовательность неизменяемых объектов) - (1, 2.0, 'Hello')
7. set 'Множетсва' (последовательность уникальных объектов) - {'a', 'b'}
8. bool 'Булевые значения' (логические значения) - true or false
